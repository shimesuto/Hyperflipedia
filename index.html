How to submit (Discord)

Please DM @symest on Discord with the information below! Sorry, but this site is static and doesn't accept direct uploads.
Back
What to include
  • Name — Self explanatory, hehe
  • Profile Link — your Discord tag (e.g. Username#1234) or a link to your main profile.
  • At least one track link — a direct track URL (SoundCloud, Bandcamp, YouTube) demonstrating the sound.
  • Short description — 1–3 sentences describing style or notable details.
  • Other necessary things — If you are not filling all the fields of a page, don't submit it.
  • Don't be shy — If you're confused, please don't be scared to ask me for help!
Discord message template
Copy & paste this template into a DM to @symest, then attach images if needed!
Name: [Name here]
Profile: [URL]
Owner (if alt): [URL]
Track example: [URL]
Description: [1–3 sentences]
Other attachments: Things like images, stats, etc.
Quick tips
  • Include working URLs — broken links make it harder to verify details!
  • If an alt is private, include screenshots or public clues that support the listing! No leaking secret alts!
  • Make sure to keep the description short and factual
  • The listing owner may follow up on Discord for clarification if needed!
